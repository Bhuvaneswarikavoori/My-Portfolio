Neuroevolution of Google Dino Game

This project is an implementation of the Google Dino Game using Python, Pygame, and the NEAT (NeuroEvolution of Augmenting Topologies) algorithm, with the objective to train an AI agent to play the game and avoid obstacles by learning to jump over cacti, where a virtual dinosaur runs through a landscape encountering various cacti as obstacles and the AI agent learns to control the dinosaur by making well-timed jumps to avoid collisions.
